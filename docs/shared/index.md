# @zenncore/shared Packages

Shared packages for both web and mobile applications.

## Packages

| Package | Description |
|---------|-------------|
| [Utils](./utils/index.md) | Utilities, hooks, and type helpers |
| [Icons](./icons/index.md) | Icon components |
| [Phone](./phone/index.md) | Phone number utilities |
| [Data Table](./data-table/index.md) | Table context and hooks |
| [Inferred Form](./inferred-form/index.md) | Schema-based form generation |

---

## Installation

```bash
bun add @zenncore/utils @zenncore/icons @zenncore/phone @zenncore/data-table @zenncore/inferred-form
```
